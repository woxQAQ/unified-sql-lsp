# Internal Packages

This directory contains private implementation packages that are not intended for external use.

## Subdirectories

- `lsp/`: LSP server implementation and protocol handlers
- `addon/`: Add-on manager and loader for Wasm plugins
- `wasm/`: Wasm runtime integration and host functions
- `parser/`: Tree-sitter parser integration and incremental parsing
- `completion/`: Completion engine and context analysis
- `connection/`: Database connection manager and pooling
- `schema/`: Schema cache management and introspection
- `config/`: Server and client configuration handling
